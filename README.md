# oop💸

A clean, modern **Expense Tracker** web application built with HTML, Tailwind CSS, and  JavaScript.  

## Features

- Add a new expense with **description**, **amount**, and **category**
- View all expenses in a structured, responsive list
- Remove (delete) any expense
- Calculate and display **total expenses** for the day
- Filter expenses by category using a dropdown
- Data persists across browser sessions using **localStorage**
- Responsive and mobile-friendly UI styled with **Tailwind CSS**

## Technologies & Concepts Used

- **HTML5** — Semantic structure
- **Tailwind CSS** — Utility-first styling & responsive design
- ** JavaScript **  
  - OOP: `Expense` and `ExpenseTracker` classes  
  - Destructuring for object property extraction  
  - Functional programming: `map`, `filter`, `reduce`  
  - Spread (`...`) & rest operators for array/object manipulation  
- **localStorage** — Persistent data storage
- **Git** — Version control with meaningful commits
- **GitHub Pages** — Live deployment
